## Getting Started

开发运行服务

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```


## 如何跑起来
修改 env 文件中的mysql 账号与密码，则可以里面跑起来
```bash
DATABASE_URL="mysql://[数据库用户名]:[数据库密码]@localhost:3306/[数据库名称]?connect_timeout=300"
```
